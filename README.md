## Testing
Running `grunt test` will run the unit tests with karma.

## To check Node and NPM version
node --version && npm --version

## Grunt commands to execute
* npm install -g grunt
* npm install -g grunt-cli
* npm install -g bower
* npm install
* bower install
* grunt build
